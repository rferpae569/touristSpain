# TouristSpain

Esta página está hecha con la idea de que sea una página de Turismo de España. Al entrar, lo que podremos hacer es seleccionar la comunidad autónoma que deseemos y ver los monumentos y los restaurantes más emblemáticos de las ciudades de dicha comunidad (Actualmente solo disponibles los monumentos de Sevilla de la comunidad de Andalucia).

Con el tiempo, se añadirá también los siguientes apartados:

1. Hoteles para la ciudad de Sevilla.
2. Información sobre las diferentes ciudades andaluzas y el resto de comunidades autonomas.
3. Un método responsive para poder abrir la página desde el móvil.

Este proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli) versión 16.0.4.
y ha sido desarrollado por Rubén Fernández.
